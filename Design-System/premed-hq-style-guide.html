Premed HQ · Design System

The canonical tokens + component rules. If a screen disagrees with this page, the screen is wrong. Fonts: Baloo 2 (display) · Nunito (body) — never change.

1 · Color tokens

Never hardcode hex in TSX. Use the CSS vars (var(--primary)…). Section colors are now first-class tokens: --sec-bb/--sec-cp/--sec-cars/--sec-ps — SECTION_META must mirror these.

2 · Radius — three roles only

Outer surfaces (cards, panels, banners) = rounded-2xl · nested items (rows, tiles, inputs-in-cards) = rounded-xl or rounded-lg · pills/avatars = rounded-full. Banned: rounded-3xl, rounded-sm, arbitrary rounded-[…].

3 · Elevation — two shadows, that's it

Resting cards use the .card-soft utility (already in index.css). Overlays (dialogs, popovers, dropdowns) use shadow-lg. Banned: shadow-sm/md on cards and one-off shadow-[…] values.

card-soft — resting card
shadow-lg — overlay/popover

Big numbers (scores, counts, countdowns) always get font-display — a text-2xl font-extrabold numeral in Nunito is a drift bug.

Weight discipline: Baloo 2 at font-extrabold (800) is reserved for the page/hero title and hero numerals — one per view. Card titles cap at font-bold (700). Extrabold everywhere is what tips a rounded font from cozy into kiddie.

Tabular figures: .font-display now sets tabular-nums automatically; live numerals in the body font (tables, timers, stat rows) take the .nums utility (or Tailwind's tabular-nums) so digits keep a fixed width and don't jitter as values change.

5 · Chips — use Badge, kill ad-hoc pills

The Badge component already has every variant needed. The 60+ hand-rolled rounded-full px-2 … spans must migrate. Section chips: <Badge style={{background:'var(--sec-bb)'}}> pattern (solid, white text).

6 · Buttons — one primary per view

Hierarchy: primary (one per view, the main action) → outline (secondary) → ghost (tertiary/inline) → destructive (confirm first). Card actions consolidate into one ⋯ overflow menu — never a row of icon buttons.

8 · Do / Don't

Do

• Tokens for every color; SECTION_META reads --sec-*.

• card-soft on every resting card.

• Gradients only as scrims over banner art or the one hero.

• Spacing rhythm: card padding p-4, section gaps gap-4/6, page stack space-y-4.

• One mascot moment per page, max.

Don't

• No hex literals in TSX (exception: AppErrorBoundary inline styles — but keep them warm-toned).

• No rounded-3xl, shadow-[…], or cold Tailwind grays (#e5e7eb, #1f2937…).

• No hand-rolled pills — Badge only.

• No decorative panel gradients or purple-blue AI defaults.

• No explainer blurbs under titles; no "coming soon" panels.
