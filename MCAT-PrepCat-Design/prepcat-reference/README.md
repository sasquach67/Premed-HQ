# PrepCat reference screenshots

Visual reference for the "MCAT PrepCat-parity" build (see `../mcat-mistake-map-CODEX-PROMPT.md`).
These are screenshots of PrepCat (prepcat.net), the MCAT-prep app the MCAT page is modeled on.
**They are visual direction only — match layout, hierarchy, and interaction patterns, not exact
colors or fonts. Premed HQ keeps its own warm-paper Ghibli theme (Baloo 2 / Nunito, `SECTION_META`
colors).**

## How to name your drops
Drop PNG/JPG screenshots into this folder. Name them `tab__state.png` so the mapping is obvious.
Suggested (rename to whatever you actually capture):

| Filename | What it should show |
|---|---|
| `dashboard__filled.png` | Dashboard with real data — readiness ring, projected +delta, section bars, right-rail tiles, heatmap |
| `dashboard__diagnostic-modal.png` | The "Add your diagnostic score" modal (4 section inputs → total) |
| `dashboard__full-length-modal.png` | The "Log a full length" modal (steppers → total → recalibrate) |
| `plan__weeks.png` | Plan, Weeks view — day columns of session cards, phases, rest, exam-day |
| `plan__months.png` | Plan, Months calendar view |
| `content__library.png` | Content grid with search + count pills + filters |
| `content__guide-detail.png` | A guide detail page — Guide/Cheat-sheet toggle, TOC, Key essentials, Trap/Figure callouts |
| `content__cheat-sheet.png` | The numbered "60-Second Night-Before Cheat Sheet" view |
| `mistakes__filled.png` | Mistake map with real mistakes plotted (the populated look) |
| `mistakes__log-modal.png` | The "Log a mistake" modal (section, topic, subtopic, you-picked/correct, type chips) |
| `stats__filled.png` | Stats with real data — projected "you are here" line, dual bars, heatmap, most-missed |
| `advisor.png` | Advisor chat with quick-reply chips |

Anything you can capture in a **filled/populated** state is the most valuable — the prompt already
describes the empty states from Claude's tour; screenshots fill in what real data looks like.
