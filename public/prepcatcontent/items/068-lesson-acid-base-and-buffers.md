# Acid base & buffers

Henderson Hasselbalch, pKa, and the titration landmarks the MCAT tests on every form.

WHAT YOU'LL LEARN

Use Henderson Hasselbalch without flipping the ratio
Know where a buffer actually buffers
Read a titration curve cold

CONCEPTS

Henderson Hasselbalch

pH = pKa + log([A⁻]/[HA]), conjugate BASE over weak ACID. When [A⁻] = [HA] the log term is 0, so pH = pKa. That single identity unlocks most buffer questions.

A buffer resists pH change

A buffer is a weak acid plus its conjugate base in comparable amounts. It works best within about ±1 pH unit of the pKa; max buffering capacity is exactly at pH = pKa, where the two forms are equal.

Titration landmarks

Half-equivalence point: half the acid is neutralized, [A⁻] = [HA], so pH = pKa. Equivalence point: moles base = moles acid. For a weak acid + strong base the equivalence pH is > 7 (the conjugate base hydrolyzes); strong + strong gives exactly 7.

Ka and Kb are linked

For a conjugate pair, Ka · Kb = Kw = 1×10⁻¹⁴, so pKa + pKb = 14. A lower pKa means a stronger acid (more dissociated). Don't confuse the endpoint (indicator color flip) with the true equivalence point.

THE RULE

At the half-equivalence point, pH = pKa. It is the fastest way to read a pKa straight off a titration curve.

CHEAT SHEET

pH = pKa + log([A⁻]/[HA]), base over acid
pH = pKa when [A⁻] = [HA] (half-equivalence, max buffer)
Buffer range ≈ pKa ± 1
Weak acid + strong base → equivalence pH > 7
Strong acid + strong base → equivalence pH = 7
Ka · Kb = Kw = 1×10⁻¹⁴ · pKa + pKb = 14 · lower pKa = stronger acid

COMMON TRAPS

·
Flipping the H H ratio (it's base/acid, not acid/base).
·
Assuming every equivalence point sits at pH 7 (only strong + strong does).
·
Putting max buffer capacity at the equivalence point, it's at half-equivalence.
·
Confusing the indicator endpoint with the true equivalence point.
