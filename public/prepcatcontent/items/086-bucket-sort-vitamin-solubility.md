Vitamin Solubility

0/12 · 0:00

0 pts

Sort each vitamin into the correct solubility class. Remember the fat-soluble vitamins with the mnemonic "ADEK", everything else (the B-complex and C) is water-soluble.

Water-soluble

B-complex and vitamin C; excreted in urine, not stored long-term

Fat-soluble

Vitamins A, D, E, K; stored in liver/fat, absorbed with dietary lipids
