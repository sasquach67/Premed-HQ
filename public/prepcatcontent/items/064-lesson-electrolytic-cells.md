# Electrolytic cells

The cell type the MCAT loves to flip on you, same reactions, opposite electrode signs.

WHAT YOU'LL LEARN

Tell galvanic vs electrolytic cells apart instantly
Assign electrode signs correctly in each
Use Q = nF and t = Q/I for plating problems

CONCEPTS

Reduction is always at the cathode

RED-CAT: reduction happens at the cathode in BOTH cell types. Oxidation is always at the anode (AN-OX). What changes between cell types is the SIGN of each electrode, never the chemistry.

Galvanic = spontaneous, makes current

A galvanic (voltaic) cell runs a spontaneous redox reaction (ΔG < 0, E°cell > 0) to push electrons out. Cathode is +, anode is −.

Electrolytic = forced by a battery

An electrolytic cell uses an external power source to drive a non-spontaneous reaction (ΔG > 0). The battery shoves electrons into the cathode, making the cathode the NEGATIVE terminal; the anode is +.

Stoichiometry of plating

Charge Q = n·F (n = mol of electrons, F = 96,500 C/mol). Time t = Q / I. Always write the half-reaction first so the electrons-per-ion factor (e.g. Cu²⁺ + 2e⁻ → Cu needs 2) is explicit.

THE RULE

Galvanic: cathode (+), anode (−). Electrolytic: cathode (−), anode (+). Reduction is at the cathode in both.

CHEAT SHEET

RED-CAT (reduction, cathode) · AN-OX (anode, oxidation), both cell types
Galvanic: spontaneous, E°cell > 0, cathode +
Electrolytic: forced, E°cell < 0, cathode −
Molten salt → reduce the metal cation; aqueous → water may reduce to H₂ instead
Q = nF, t = Q/I, moles e⁻ = (mass/molar mass) × n

COMMON TRAPS

·
"Cathode is positive", only true for galvanic cells.
·
Forgetting the electrons-per-ion factor (Cu²⁺ needs 2 e⁻) and being off by 2×.
·
Treating molten NaCl like aqueous NaCl (water isn't there to reduce).
